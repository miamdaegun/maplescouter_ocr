# 메이플 북마크 추가기 with Gemini - Chrome 확장 프로그램

메이플스토리 아이템 툴팁 스크린샷을 Gemini AI로 OCR 분석하여 maplescouter.com의 북마크(`equipBookmarkList`)에 자동으로 추가하는 Chrome 확장 프로그램입니다.

## 주요 기능

- **AI 스크린샷 분석**: Google AI Studio(Gemini) API로 아이템 툴팁 이미지를 자동 파싱
- **모델 선택**: Gemini 2.5 Flash (하루 20회) / Gemini 3.1 Flash-Lite (하루 500회)
- **모델 추천**: 3.1 Flash-Lite 모델을 추천함(기본적으로 2.5 보다 더 빠름, 하루 무료 요청 횟수 제한 역시 많음), 대부분 정확하게 인식하나 가끔 찐빠날 때가 있긴함 (2.5는 잘 인식하는데 3.1은 가끔 인식 못함)
- **3.1 특징**: 외국 형님들이 일어나시는 저녁시간대 부터 3.1은 혼잡한 경우가 많음 (어쩔수 없이 2.5 사용 추천)
- **일일 할당량 추적**: 무료 API 사용 횟수를 자동으로 카운팅
- **결과 수정**: OCR 파싱 결과를 추가 전에 직접 수정 가능
- **북마크 목록 확인**: 현재 저장된 북마크 목록 조회
- **세션 유지**: 마지막 작업 상태 자동 저장 및 복원

## 설치 방법

0. github 사이트에서 초록색의 <>code 버튼 클릭 후 하단의 Download zip 클릭, 압축 해제
1. Chrome 브라우저에서 `chrome://extensions` 주소로 이동합니다.
2. 우측 상단 **개발자 모드** 토글을 켭니다.
3. **압축 해제된 확장 프로그램을 로드합니다** 버튼 클릭
4. 압축 푼 폴더를 선택합니다.

## 사용 방법

1. **https://maplescouter.com/ko/item** 에 접속합니다.
2. 브라우저 우측 상단 확장 프로그램 아이콘을 클릭합니다.
3. 최초 1회 **Gemini API 키**를 입력하고 저장합니다. ([aistudio.google.com](https://aistudio.google.com)에서 무료 발급)
4. 좌하단의 get API key 클릭 후 API Key 복사 (API Key가 없다면 해당 페이지에서 API 키 만들기 클릭)
5. 그 후 주소창 오른쪽의 해당 아이콘 클릭 후 프로그램 실행, 사용할 **모델**을 선택합니다.
<img src=icons\test3.png>

6. 아이템 툴팁 스크린샷을 준비합니다. (Win+Shift+S 캡처 후 Ctrl+V 붙여넣기 또는 파일 드래그)
7. **AI로 아이템 옵션 읽기** 버튼을 클릭합니다. 이미지 양식은 다음과 같습니다. 네모박스에 정확히 딱 맞게 스크린샷을 찍으면 됩니다.
<img src=icons\test.png>

8. 파싱 결과를 확인하고 필요 시 수정 후 **북마크에 추가** 버튼을 클릭합니다.
9. 그 후 아이템 메이커 창에서 새로고침 후 제대로 등록이 됐는지 확인합니다.

## 참고 사항
- 요구 레벨, 스타포스 수치, 주문서 작 횟수 등 환산에 필요없는 부분은 실제 아이템과 ocr 파싱 결과가 달라도 상관없습니다! (지금까지 테스트할땐 문제 없었음)
- 아이템 메이커에 추가 후 환산에서 추가한 장비의 ui를 보면 기본 수치가 흰색 수치만 보일텐데 실제 환산에 반영될 땐 총합 토탈만 보면 되기 때문에 계산이 제대로 작동할 겁니다.

## 주의사항

- **해당 프로그램을 이용해 등록한 아이템에 문제가 생길 시 환산에 문의하지 마시고 본인이 직접 수기로 등록해서 이용해주세요.**
- https://maplescouter.com/ko/item 페이지 상에서 해당 확장 프로그램을 실행해야 북마크 추가/조회가 가능합니다.
- Gemini 무료 API는 일일 사용 횟수 제한이 있습니다. 초과 시 다음 날 리셋됩니다.
- 아이템의 이미지까지 추출하진 못합니다.
- 말도 안되는 이미지 분석 요청하면 ai가 뻗어버릴 수도 있습니다.
- 이상하게 아이템 만들어서 등록한 뒤 환산에서 해당 아이템으로 비교, 검색 등 하면 해당 아이템으로 오류가 생겨 하드 리셋 해야할 수도 있습니다.
- 무료 버전에서는 분당 요청 횟수 제한도 존재합니다. 너무 많이 요청할 시 제한이 걸리므로 천천히 해주세요.
- 무료 버전의 할당량을 모두 사용한 경우 유료 버전을 이용하면 됩니다. 계산 상으로 하루 100번 요청을 한달동안 매일 했을 때 3000~4000원 정도로 예상됩니다.
- 유료 버전 이용 시 분당 한도도 늘어나게 됩니다. 자세한 사항은 aistudio의 제한량을 참고해주세요.
- 버그나 제안할 사항은 인벤 '미암대군' 쪽지 또는 miamdaegun@gmail.com 으로 반드시 알려주세요.
- 프롬프트 변경 후 혹시 더 잘된다/ 더 빠르다 싶은 것도 저한테 알려주시면 감사하겠습니다.
- api 남은 횟수를 실시간으로 불러오는게 아니여서 정확하지 않을 수 있습니다. 정확한 내용은 aistudio 사이트에서 직접 확인해주세요.
- 본인 귀책으로 인한 api 정지 등에 대해 책임을 지지 않습니다.

개인정보 처리방침

환산 아이템 추가 프로그램(이하 "확장 프로그램")는 사용자의 개인정보를 소중히 여깁니다.

수집하는 정보

  • Gemini API 키: 사용자가 직접 입력한 키를 기기 내 chrome.storage.local에만 저장하며, 외부 서버로 전송되지 않습니다.
  • API 사용 횟수: 무료 할당량 관리를 위해 총 호출 횟수를 Firebase(Firestore)에 익명으로 기록합니다. 이 데이터는 특정 사용자를 식별하지 않습니다.
  • 아이템 이미지: OCR 분석을 위해 업로드한 이미지는 Google Gemini API로 전송됩니다. 이미지는 서버에 저장되지 않으며 분석 후 즉시 파기됩니다.

수집하지 않는 정보

  • 이름, 이메일 등 개인 식별 정보
  • 브라우저 히스토리, 위치 정보
  • API 키 외의 계정 정보

제3자 서비스

  • Google Gemini API: 이미지 분석에 사용됩니다. Google 개인정보 처리방침이 적용됩니다.
  • Firebase (Firestore): 익명 사용 통계 저장에 사용됩니다. Firebase 개인정보 처리방침이 적용됩니다.
